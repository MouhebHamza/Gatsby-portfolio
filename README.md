## Stacks

- Gatsby js
- Styled Components

## Installation

- Clone repository
  <br>
  ```
  git clone https://github.com/MouhebHamza/Gatsby-portfolio.git
  ```
- Navigate into directory and install dependencies
  ```
  cd gatsby-portfolio
  npm install
  ```
- Start it up
  ```
   gatsby develop
  ```
